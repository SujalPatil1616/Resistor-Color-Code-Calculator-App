# Resistor Color Code Calculator App

## Overview

The Resistor Color Code Calculator is an Android application developed using MIT App Inventor. The application helps users determine the resistance value of a resistor by selecting its color bands.

Instead of manually referring to resistor color code charts, users can simply select the first digit, second digit, multiplier, and tolerance bands to instantly calculate the resistor value and tolerance.

This application is designed as a quick educational tool for Electronics and Communication Engineering (ECE) students, hobbyists, and beginners in circuit design.

---

## Introduction

Resistors are one of the most commonly used electronic components. Their resistance values are represented using color bands printed on the resistor body.

The application simplifies the process of decoding resistor values by automatically converting selected color bands into their corresponding resistance value.

## Resistor Color Code Theory

A standard four-band resistor consists of:

### First Band
Represents the first significant digit.

### Second Band
Represents the second significant digit.

### Third Band
Represents the multiplier value.

### Fourth Band
Represents the tolerance percentage.

Example:

Brown = 1

Red = 2

Orange = ×10³

Silver = ±10%

Resistance:

12 × 10³ Ω

= 12,000 Ω ±10%

## Working Principle

The application allows users to select resistor color bands from dropdown menus.

Each selected color corresponds to a predefined numerical value stored within the application. When the Calculate Resistance button is pressed, the app combines the first two digits, applies the multiplier value, and displays the final resistance along with the selected tolerance.

The entire calculation process is performed instantly using MIT App Inventor logic blocks.
